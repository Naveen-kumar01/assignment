The fbprophet is library developed by facebook for the multivariate forecasting only-
The requirement to run this are -
install pystan first by using cmd- pip install pystan==2.19.1.1
Then we have to install fbprophet using cmd - conda install -c conda-forge fbprophet
The fbprophet and pystan are not compatible with latest version of python there are lot of dependency so -
1. python- 3.7 or 3.6.12
2. numpy- 1.19
3. pystan- 2.19.1.1
4. fbprophet- 0.7.1 or 0.6
5. windows 10 version 19410 and above
